# 1️⃣ `getOrDefault()` (MOST USED in DSA)

### What it does

Returns value if key exists, otherwise returns **default value**

### Syntax

```java
map.getOrDefault(key, defaultValue)
```

### Example

```java
Map<Integer,Integer> map = new HashMap<>();

int count = map.getOrDefault(5, 0);
```

If key `5` not present → returns `0`

---

### DSA Frequency Count (most common use)

```java
map.put(num, map.getOrDefault(num,0)+1);
```

Instead of:

```java
if(map.containsKey(num))
   map.put(num, map.get(num)+1);
else
   map.put(num,1);
```

---

# 2️⃣ `putIfAbsent()`

### What it does

Put value **only if key is NOT present**

### Syntax

```java
map.putIfAbsent(key, value);
```

### Example

```java
map.putIfAbsent(1, 100);
map.putIfAbsent(1, 200);
```

Result:

```
1 → 100
```

Second call ignored.

---

### When to use

Initialize only once

Example

```java
map.putIfAbsent(key, new ArrayList<>());
```

---

# 3️⃣ `computeIfAbsent()` (VERY IMPORTANT)

### What it does

If key absent → create value using function

### Syntax

```java
map.computeIfAbsent(key, k -> new ArrayList<>());
```

---

### Example (grouping problem)

```java
Map<Integer,List<Integer>> map = new HashMap<>();

map.computeIfAbsent(1, k -> new ArrayList<>()).add(10);
```

Equivalent to:

```java
if(!map.containsKey(1))
   map.put(1,new ArrayList<>());

map.get(1).add(10);
```

---

### When to use

Grouping problems

* group anagrams
* graph adjacency list
* bucket grouping

---

# 4️⃣ `poll()` vs `remove()`

Used in Queue / Deque

## poll()

* removes head
* returns null if empty
* safe

```java
q.poll();
```

---

## remove()

* removes head
* throws exception if empty

```java
q.remove();
```

---

### Interview answer

poll → safe
remove → throws exception

---

# 5️⃣ `offer()` vs `add()`

Both insert into queue

## offer()

* returns false if fail
* safe

```java
q.offer(10);
```

---

## add()

* throws exception if fail

```java
q.add(10);
```

---

Interview rule:
Use **offer** in queues

---

# 6️⃣ `peek()` vs `element()`

Both read head of queue

## peek()

* returns null if empty
* safe

```java
q.peek();
```

---

## element()

* throws exception if empty

```java
q.element();
```

---

# ⭐ MEMORY TABLE

| Method  | Safe | Throws exception |
| ------- | ---- | ---------------- |
| poll    | yes  | no               |
| remove  | no   | yes              |
| offer   | yes  | no               |
| add     | no   | yes              |
| peek    | yes  | no               |
| element | no   | yes              |

---

```java
map.getOrDefault()

```

# ⭐ MOST IMPORTANT EXAMPLE (GROUPING)
Using putIfAbsent
```java
map.putIfAbsent(key, new ArrayList<>());
map.get(key).add(value);
```
Using computeIfAbsent (cleaner)
```java
map.computeIfAbsent(key, k -> new ArrayList()).add(value);
```

This is why computeIfAbsent is preferred

## 📦 Arrays & Collections

### Arrays & Collections — lifecycle oriented (creation → transform → search → compare → utility)

---

## Creation / Initialization

### Arrays

* fill entire array → `Arrays.fill(arr, value)`
* fill range → `Arrays.fill(arr, start, end, value)`
* resize / clone array → `Arrays.copyOf(arr, newLength)`
* copy sub range → `Arrays.copyOfRange(arr, start, end)`

### Collections

* fill list → `Collections.fill(list, value)`
* copy list → `Collections.copy(dest, src)`
* make read only → `Collections.unmodifiableList(list)`
* thread safe list → `Collections.synchronizedList(list)`

---

## Ordering / Transformation

### Arrays

* sort primitive array → `Arrays.sort(arr)`
* sort with comparator → `Arrays.sort(objArr, comparator)`
* parallel sorting → `Arrays.parallelSort(arr)`

### Collections

* sort list → `Collections.sort(list)`
* reverse list → `Collections.reverse(list)`
* randomize order → `Collections.shuffle(list)`
* rotate elements → `Collections.rotate(list, distance)`
* swap elements → `Collections.swap(list, i, j)`
* replace values → `Collections.replaceAll(list, oldVal, newVal)`

---

## Searching / Fetching

### Arrays

* search in sorted array → `Arrays.binarySearch(arr, key)`
* search in range → `Arrays.binarySearch(arr, start, end, key)`

### Collections

* search sorted list → `Collections.binarySearch(list, key)`
* count occurrences → `Collections.frequency(list, value)`
* find minimum → `Collections.min(list)`
* find maximum → `Collections.max(list)`
* no common elements → `Collections.disjoint(list1, list2)`

---

## Comparison

### Arrays

* compare 1D arrays → `Arrays.equals(a, b)`
* compare 2D arrays → `Arrays.deepEquals(a, b)`

---

## Debug / Printing

### Arrays

* print 1D array → `Arrays.toString(arr)`
* print 2D array → `Arrays.deepToString(arr)`

### Collections

* print list → `System.out.println(list)`
* print set → `System.out.println(set)`
* print map → `System.out.println(map)`
* print nested collections → `System.out.println(collection)`

---

## Conversion / Stream

### Arrays

* convert to stream → `Arrays.stream(arr)`

### Collections

* convert to stream → `collection.stream()`

---
